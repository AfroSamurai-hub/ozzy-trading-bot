## 🚀 COPY/PASTE COMMANDS

### Test USD (Default - No Changes):
```bash
cd ~/ozzy-simple && source venv/bin/activate && python scripts/test_currency.py
```

### Test ZAR:
```bash
cd ~/ozzy-simple && source venv/bin/activate && \
OZZY_CURRENCY=ZAR OZZY_USD_TO_ZAR=18.50 python scripts/test_currency.py
```

### Run Trading Bot with ZAR:
```bash
cd ~/ozzy-simple && source venv/bin/activate && \
export OZZY_CURRENCY=ZAR && \
export OZZY_USD_TO_ZAR=18.50 && \
python scripts/test_live_stream.py --symbol BTCUSDT --duration 1800
```

### Run Validation Test with ZAR:
```bash
cd ~/ozzy-simple && source venv/bin/activate && \
export OZZY_CURRENCY=ZAR && \
export OZZY_USD_TO_ZAR=18.50 && \
python scripts/run_validation_test.py
```

### Make ZAR Permanent (Add to .env):
```bash
echo "OZZY_CURRENCY=ZAR" >> ~/ozzy-simple/.env
echo "OZZY_USD_TO_ZAR=18.50" >> ~/ozzy-simple/.env
```
