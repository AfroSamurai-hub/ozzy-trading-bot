# MOMENTUM STRATEGY - Evidence-Based Trading Rules

## Peer-Reviewed Foundation
**Source:** Jegadeesh & Titman (1993), Journal of Finance  
**Replication:** 30+ years out-of-sample validation  
**Our Validation:** Backtest showed 43.8% WR without filters → Need confirmations!

## Definition
Buy assets with strong 3-12 month performance relative to recent baseline.

## Expected Performance (REALISTIC - Not Marketing BS)
- **Win Rate:** 55-60% (NOT 84% - that's marketing)
- **Monthly Return:** 1-1.5%
- **Annual Return:** 12-18%
- **Sharpe Ratio:** 0.6-1.0
- **Max Drawdown:** 20-30%

## Entry Criteria (ALL must be true)
1. ✅ **Trend Confirmation:** Price above 200-day MA
2. ✅ **Momentum Threshold:** 12-month return > 15%
3. ✅ **Volume Confirmation:** Volume > 1.5x 20-day average
4. ✅ **Momentum Mode:** RSI > 50 (NOT overbought, confirming momentum)
5. ✅ **Breakout Room:** No major resistance within 5%
6. ✅ **Market Regime:** Not in high volatility regime (VIX < 30)

## Exit Criteria
1. **Stop Loss:** 2% below entry (ALWAYS set)
2. **Take Profit:** 4% above entry (2:1 R/R minimum)
3. **Time Stop:** Exit after 96 candles (24 hours @ 15m) if no movement
4. **Regime Exit:** Exit ALL positions if VIX > 40

## When Strategy FAILS (Learn from History)
- **Market Crashes:** Momentum reverses suddenly (see COVID March 2020)
- **Ranging Markets:** Whipsaws increase (see 2015-2016)
- **Crowding:** When everyone uses momentum (see 2017 momentum crash)
