## 🎯 HOW TO USE

### Quick Start - USD (Default):
```bash
cd ~/ozzy-simple
source venv/bin/activate
python scripts/test_live_stream.py --symbol BTCUSDT --duration 1800
```

### Quick Start - ZAR:
```bash
cd ~/ozzy-simple
source venv/bin/activate
export OZZY_CURRENCY=ZAR
export OZZY_USD_TO_ZAR=18.50
python scripts/test_live_stream.py --symbol BTCUSDT --duration 1800
```

### Permanent ZAR (Add to .env):
```bash
# Edit ~/ozzy-simple/.env and add:
OZZY_CURRENCY=ZAR
OZZY_USD_TO_ZAR=18.50
```

## 📞 SUPPORT

### If USD Mode Doesn't Work:
```bash
# Check you haven't set ZAR accidentally:
echo $OZZY_CURRENCY
# Should be empty or "USD"

# Unset if needed:
unset OZZY_CURRENCY
```

### If ZAR Mode Doesn't Work:
```bash
# Verify environment variables:
echo $OZZY_CURRENCY  # Should show: ZAR
echo $OZZY_USD_TO_ZAR  # Should show: 18.50 (or your rate)

# Set if missing:
export OZZY_CURRENCY=ZAR
export OZZY_USD_TO_ZAR=18.50
```

### If Exchange Rate Needs Updating:
```bash
# Get latest rate from Google or XE.com
# Then update:
export OZZY_USD_TO_ZAR=18.75  # Example new rate

# Or edit .env file:
nano ~/ozzy-simple/.env
# Change: OZZY_USD_TO_ZAR=18.75
```
