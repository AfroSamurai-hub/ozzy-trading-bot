## Configuration Options ⚙️

### Change Summary Frequency:
Edit `scripts/test_live_stream.py` line ~82:
```python
SUMMARY_INTERVAL = 300  # 300 = 5 minutes
```

Change to:
- `180` = 3 minutes (more frequent)
- `600` = 10 minutes (less frequent)
- `900` = 15 minutes (way less frequent)

### Change Alert Thresholds:
Edit `scripts/test_live_stream.py` line ~235:
```python
if pnl_pct >= 2.0 or pnl_pct <= -1.0:  # Current
```

Change to:
- `>= 1.5 or <= -0.8` = Earlier alerts (more frequent)
- `>= 2.5 or <= -1.2` = Later alerts (less frequent)
