## 📊 LIVE TICKER SHOWS

- **Capital:** Current available capital (updates in real-time)
- **P&L:** Total profit/loss in $ and % (green if positive, red if negative)
- **Positions:** Number of open positions
- **Trades:** Number of closed trades
- **Win Rate:** Overall win percentage (only shows when trades exist)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

## 🚀 HOW TO SEE IT

Restart your dashboard:

```bash
cd ~/ozzy-simple
source venv/bin/activate
export OZZY_CURRENCY=ZAR
export OZZY_USD_TO_ZAR=18.50
python dashboard/cli_monitor.py
```

The headline will:
- ✅ Update every 5 seconds
- ✅ Show live P&L changes
- ✅ Track positions opening/closing
- ✅ Display win rate as trades complete
- ✅ Color-code profits (green) and losses (red)
