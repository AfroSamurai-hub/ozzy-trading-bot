- GPT-4 agent → Unpredictable decisions

**Our Own Research Said:**

**The Truth:**

**Fantasy:**
- R10K → R5K/week profit (50%/week = 2,600%/year)

**Reality:**

R10K cannot generate R5K/week. That requires 50%/week returns, which is institutional-level performance with orders of magnitude more capital and infrastructure.

### **Lesson 6: Transaction Costs Kill Most Strategies**

**Per Trade Cost on Bybit:**
- Maker fee: 0.02% (if providing liquidity)
- Taker fee: 0.055% (if taking liquidity)
- Typical: 0.1% round-trip

**Impact on R10K Account:**
- 10 trades/month × R10K × 0.1% = R100 in fees
- 50 trades/month = R500 in fees
- 80 trades/month = R800 in fees

**Reality:**
Need 8% monthly profit just to break even on 80 trades. That's 96% annualized return BEFORE profit.

**Solution:**
Trade LESS, trade BETTER. 2-4 high-quality trades > 80 mediocre trades.

---

### **Lesson 7: Indicators Don't Predict, They Lag**

**What We Built:**
- 26 chart patterns
- 10+ technical indicators
- ML models on historical patterns
- Fibonacci levels
- Support/resistance

**What We Forgot:**
All indicators are LAGGING. They tell you what happened, not what will happen.

**The Truth:**
- RSI tells you price was oversold YESTERDAY
- EMA crossover tells you trend changed YESTERDAY
- Patterns appear AFTER the move
- ML overfits to HISTORICAL data

**Better Approach:**
Use simple indicators for PROBABILITY, not PREDICTION. Accept 50-60% win rate with 2:1 R/R.

```python
def generate_signal(self, candles: List[Dict]) -> Dict:
    """Generate LONG/SHORT/SKIP signal"""
    
    # Calculate indicators
    rsi = self.calculate_rsi(closes)
    ema_short = self.calculate_ema(closes, 20)
    ema_long = self.calculate_ema(closes, 50)
    
    # Filter volume outliers (IQR method)
    volumes_filtered = self._filter_volume_outliers(volumes[-20:])
    avg_volume = np.mean(volumes_filtered)
    
    # Volume confirmation
    volume_ratio = current_volume / avg_volume
    volume_confirmed = volume_ratio > self.volume_mult
    
    # LONG conditions (need ≥30% confidence)
    long_conditions = [
        rsi < self.rsi_oversold,      # RSI < 40
        ema_short > ema_long,          # Uptrend
        current_price > ema_short,     # Above trend line
        volume_confirmed               # Volume spike
    ]
    
    # SHORT conditions
    short_conditions = [
        rsi > self.rsi_overbought,     # RSI > 60
        ema_short < ema_long,          # Downtrend
        current_price < ema_short,     # Below trend line
        volume_confirmed               # Volume spike
    ]
    
    # Calculate confidence
    long_score = sum(long_conditions) / 4 * 100
    short_score = sum(short_conditions) / 4 * 100
    
    # Return best signal if >= 30%
    if long_score >= self.min_confidence:
        return {'signal': 'LONG', 'confidence': long_score, ...}
    elif short_score >= self.min_confidence:
        return {'signal': 'SHORT', 'confidence': short_score, ...}
    else:
        return {'signal': 'SKIP', 'confidence': 0, ...}
```

```python
def calculate_position_size(self, entry_price: float, stop_loss: float) -> Dict:
    """Calculate position size using 1% risk rule"""
    
    risk_amount = self.capital * self.risk_per_trade  # 1% of capital
    risk_per_unit = abs(entry_price - stop_loss)      # Distance to stop
    position_size = risk_amount / risk_per_unit       # Units to trade
    
    return {
        'size': position_size,
        'value': position_size * entry_price,
        'risk': risk_amount
    }

def pre_trade_checks(self, signal: Dict) -> Tuple[bool, str]:
    """Verify trade allowed by risk rules"""
    
    # Check 1: Max positions (2 max)
    if len(self.open_positions) >= self.max_positions:
        return False, "Max positions reached"
    
    # Check 2: Portfolio heat (<6% total exposure)
    total_heat = sum(p['risk'] for p in self.open_positions)
    if total_heat + signal['risk'] > self.max_heat:
        return False, "Portfolio heat exceeded"
    
    # Check 3: Daily loss limit (<3% loss today)
    if abs(self.daily_pnl) > self.daily_loss_limit * self.capital:
        return False, "Daily loss limit hit"
    
    return True, "Approved"
```

### **src/bybit_client.py (265 lines)**

```python
def get_candles(self, symbol="BTCUSDT", interval="240", limit=100):
    """Fetch historical candles from Bybit V5 API"""
    
    response = self.session.get_kline(
        category="linear",
        symbol=symbol,
        interval=interval,  # "240" = 4-hour
        limit=limit
    )
    
    candles = []
    for kline in response['result']['list']:
        candles.append({
            'timestamp': int(kline[0]),
            'open': float(kline[1]),
            'high': float(kline[2]),
            'low': float(kline[3]),
            'close': float(kline[4]),
            'volume': float(kline[5]),
            'turnover': float(kline[6]) if len(kline) > 6 else 0
        })
    
    candles.reverse()  # Bybit returns newest first
    logger.info(f"📊 Fetched {len(candles)} candles for {symbol} ({interval}m)")
    return candles
```

**Main Loop:**
```python
def run(self):
    """Main trading loop"""
    self.running = True
    self.start_time = datetime.now()
    
    while self.running:
        try:
            self.decision_count += 1
            
            # Fetch candles
            interval_minutes = str(int(TIMEFRAME.replace('h', '')) * 60)
            candles = self.client.get_candles(interval=interval_minutes)
            
            if not candles:
                logger.warning("No candle data received")
                time.sleep(60)
                continue
            
            # Generate signal
            signal = self.signal_gen.generate_signal(candles)
            
            # Track stats
            self.signal_stats[signal['signal']] += 1
            if signal['confidence'] > 0:
                self.confidence_sum += signal['confidence']
                self.confidence_count += 1
            
            # Log signal
            logger.info(f"🎯 SIGNAL: {signal['signal']}")
            logger.info(f"📊 Confidence: {signal['confidence']:.1f}%")
            
            # Execute if tradeable
            if signal['signal'] in ['LONG', 'SHORT']:
                approved, reason = self.risk_mgr.pre_trade_checks(signal)
                if approved:
                    logger.info("✅ APPROVED TO TRADE")
                    logger.info("📝 PAPER MODE - Trade not executed")
                else:
                    logger.warning(f"❌ REJECTED: {reason}")
            
            # Show dashboard every 5 decisions
            if self.decision_count % 5 == 0:
                self.print_status_dashboard()
            
            # Wait 4 hours
            logger.info(f"⏰ Next check in {CHECK_INTERVAL}s (4h)...")
            time.sleep(CHECK_INTERVAL)
            
        except KeyboardInterrupt:
            logger.info("🛑 Shutdown requested")
            self.running = False
        except Exception as e:
            logger.error(f"❌ Error: {e}")
            time.sleep(60)
```

### **Position Sizing Math**

**1% Risk Rule:**
```
Risk Amount = Capital × 0.01
Stop Distance = Entry Price - Stop Loss Price
Position Size = Risk Amount / Stop Distance

Example:
Capital: R5,000
Risk: R50 (1%)
Entry: R100,000 BTC
Stop: R98,000 (2% below entry)
Distance: R2,000
Position: R50 / R2,000 = 0.025 BTC
```

**Why This Works:**
- Caps loss at 1% per trade
- Normalizes position size across volatility
- 100 losing trades = complete wipeout (very unlikely)
- Survive to learn

### **ATR (Average True Range)**

**Purpose:** Measure volatility for stop loss placement

**Calculation:**
```python
# True Range for each candle
tr = max(
    high - low,                    # Candle range
    abs(high - previous_close),    # Gap up

)

# ATR = Average of last 14 TRs
atr = mean(tr_list[-14:])
```

**Stop Loss Placement:**

**Why:** Sometimes ATR calculates unreasonably wide (67% stop loss), cap it.

### **Volume Outlier Filtering**

**Problem:** One 11.4M volume candle skews average to 1.1M (should be ~300K)

**Solution - IQR Method:**
```python
def _filter_volume_outliers(volumes: List[float]) -> List[float]:
    """Remove extreme outliers using IQR method"""
    
    volumes_arr = np.array(volumes)
    
    # Calculate quartiles
    q1 = np.percentile(volumes_arr, 25)  # 25th percentile
    q3 = np.percentile(volumes_arr, 75)  # 75th percentile
    iqr = q3 - q1                        # Interquartile range
    
    # Define outlier bounds (3× IQR)
    lower = q1 - 3 * iqr
    upper = q3 + 3 * iqr
    
    # Filter outliers
    filtered = volumes_arr[(volumes_arr >= lower) & (volumes_arr <= upper)]
    
    return filtered.tolist()
```

**Why 3× IQR:** Standard is 1.5×, but crypto has legitimate spikes. 3× catches only extreme outliers.

### **RSI (Relative Strength Index)**

**Purpose:** Identify overbought/oversold conditions

**Calculation:**
```python
def calculate_rsi(closes: List[float], period: int = 14) -> float:
    """Calculate RSI using Wilder's smoothing"""
    
    # Calculate price changes
    deltas = np.diff(closes)
    
    # Separate gains and losses
    gains = np.where(deltas > 0, deltas, 0)
    losses = np.where(deltas < 0, -deltas, 0)
    
    # Calculate average gain/loss
    avg_gain = np.mean(gains[-period:])
    avg_loss = np.mean(losses[-period:])
    
    # RSI formula
    if avg_loss == 0:
        return 100  # All gains, fully overbought
    
    rs = avg_gain / avg_loss
    rsi = 100 - (100 / (1 + rs))
    
    return rsi
```

**Interpretation:**
- **RSI < 30:** Oversold (potential bounce)
- **RSI > 70:** Overbought (potential pullback)
- **RSI 40-60:** Neutral (no clear signal)

**Our Thresholds (Relaxed for Testing):**
- Oversold: < 40 (easier to trigger)
- Overbought: > 60 (easier to trigger)

### **EMA (Exponential Moving Average)**

**Purpose:** Identify trend direction

**Calculation:**
```python
def calculate_ema(closes: List[float], period: int) -> float:
    """Calculate EMA using exponential smoothing"""
    
    prices = np.array(closes)
    
    # Start with simple moving average
    ema = np.mean(prices[:period])
    
    # Smoothing factor
    multiplier = 2 / (period + 1)
    
    # Apply exponential smoothing
    for price in prices[period:]:
        ema = (price - ema) * multiplier + ema
    
    return ema
```

**Trend Detection:**
```python
ema_short = calculate_ema(closes, 20)  # Fast EMA
ema_long = calculate_ema(closes, 50)   # Slow EMA

if ema_short > ema_long:
    trend = "UPTREND"      # Fast above slow = bullish
elif ema_short < ema_long:
    trend = "DOWNTREND"    # Fast below slow = bearish
else:
    trend = "SIDEWAYS"     # Equal = no clear trend
```
